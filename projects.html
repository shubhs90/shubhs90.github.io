My Projects

  1. Making Celery Server Highly Available in Pulp OSS

  -Designed and implemented an algorithm to make Celery server ‘highly available’.

  -Worked with RedHat's Satellite agile team to make the product HA for the next release cycle.

  -Implemented new features in the web services module using Docker development environment.

  -Developed TDD model using mock unit testing framework and wrote 100% test coverage for the module.

  2. Click Through Rate Prediction (A Kaggle Competition)

  -Implemented and analyzed the probabilistic model of a click by the user for a particular ad.

  -Implementation included analysis of split data and results using Logistic Regression(SGD Regression) and Random Forest algorithms using Python(Scikit learn) and Weka.

  -Data was analyzed using MongoDB and Python Scripts.

  -Achieved 74.2% precision using various pre-processing measures and prediction techniques.

  3. Twitter Sentiment Analyzer Using NodeJS

  -Implemented and analyzed the stream of Love/Hate Tweets using ntwitter, ExpressJS and NodeJS.

  -Handled the condition of big twitter stream data on server and client side using Node socket Module.

  -Used CanvasJS for plotting real time charts on the client browser.

  -Analysis showed that hate tweets(12-21%) increased during evening time and again decreased at night time.

  4. WolfGrader: Course Assessment System

  -Developed a course assessment application system using Java Servlets and MySQL.

  -Collaborated with team to come up with unique questionnaire based forum in the application.

  - Also implemented a add-on feature of discussion based groups in the existing system.

  -Used various features of collections framework of Java to come up with agile solutions.

  5.ELM-Web Application for Reviewing Media Files

  -Developed a back end review system for media contents using Django Web Framework, Postgresql and Python.

  -Build a contest application on top of the framework which is used to submit and review media contents for a contest.

  -Application is currently in production and used by NC State Libraries for various purposes.

  -Prepared documentation for the existing system to make collaborations easier for the future developers of this application.

  6. My Health Companion

  -Developed a web diary using Django Web Framework and Python for gathering health data of an individual.

  -Implemented an Android application which collected user health information for statistical and storage purposes using Firebase backend system.

  -Deployed the application for dumb phones which used SMS feature to collect data from the user.

  -Android application was showcased in HackDuke Hackathon event by Duke University.
